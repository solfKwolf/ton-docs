# 入门指南
在深入研究DApps之前，请确保您理解底层的区块链原理。我们的[The Open Network](/learn/introduction)和[Blockchain of Blockchains](/learn/overviews/ton-blockchain)文章可能会对您有所帮助。

TON DApps是没有后端服务，与区块链交互的应用程序。在大多数情况下，它们与自定义的[智能合约](/develop/smart-contracts/)交互；本文档提供了处理TON中可用的标准资产的方法，既可以用作示例，也可以用于更快的dapp开发。

dapp通常可以用任何拥有面向TON的SDK的编程语言编写。实际上，最常见的选择是网站，其次是Telegram的小应用程序。
